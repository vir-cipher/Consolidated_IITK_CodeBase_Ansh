# ml-phishing/ — flag phishing sites with machine learning  (my flagship)
A small decision tree that learns to tell phishing URLs from real ones using 30 URL features.
Run: `pip install scikit-learn` then `python phishing_decision_tree.py` (needs `dataset.csv`).
Dataset + idea: github.com/npapernot/phishing-detection (UCI). My experiments are mine.
